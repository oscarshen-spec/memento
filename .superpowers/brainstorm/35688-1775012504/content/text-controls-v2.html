Interaction
  • Default state: Font (Aa) is active, shows 4 font option tiles above
  • Tap Aa → upper row switches to Serif / Sans / Mono / Hand tiles
  • Tap color wheel → upper row switches to gradient strip + quick swatches; text color updates live as you drag
  • Selected font tile is white-filled; inactive tiles are dim
  • Color strip thumb slides to pick any color; swatches are quick-picks
