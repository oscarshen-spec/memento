How should it be built?

3 approaches — pick the one that feels right. I'll explain trade-offs below each.

konva canvas here

A — Pure CSS Frame

Hardcover boards + lined paper are all CSS/HTML wrapping the canvas. The Konva stage is transparent and floats on top.

Pros

  • Simple — no Konva changes
  • Easy to tweak visually

Cons

  • Paper lines won't export with scraps
Konva Layer 0: lines
Konva Layer 1: scraps

B — Hybrid ⭐ Recommended

Hardcover boards are CSS. Lined paper is a Konva background layer (Layer 0) inside the canvas. Scraps go on Layer 1.

Pros

  • Paper lines export with the page
  • Boards stay outside export
  • Clean separation

Cons

  • Small Konva refactor needed
all Konva
shapes

C — Full Konva

Everything — boards, lines, scraps — drawn inside one Konva Stage. Perfect for export fidelity.

Pros

  • Best export fidelity
  • Single rendering context

Cons

  • Most complex to build
  • Hardcover boards scale oddly on resize
